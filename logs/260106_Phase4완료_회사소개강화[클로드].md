## 📝 기술 노트

### 타임라인 컴포넌트 구현

**레이아웃 구조**:
```tsx
<div className="relative">
  {/* 중앙 세로 라인 */}
  <div className="absolute left-1/2 bg-lime-200" />

  {/* 타임라인 아이템들 (좌우 교차) */}
  <div className="space-y-12">
    {/* 왼쪽 */}
    <div className="md:justify-start">
      <div className="md:w-1/2 md:pr-8 text-right">...</div>
      <div className="absolute left-1/2 bg-lime-600 rounded-full" />
    </div>

    {/* 오른쪽 */}
    <div className="md:justify-end">
      <div className="md:w-1/2 md:pl-8">...</div>
      <div className="absolute left-1/2 bg-lime-600 rounded-full" />
    </div>
  </div>
</div>
```

**반응형 처리**:
- Mobile: 단일 컬럼 (중앙 라인 숨김)
- Desktop: 좌우 교차 배치 (지그재그)

### Gradient 배경 구현

**비전/미션 섹션**:
```tsx
<section className="bg-gradient-to-br from-lime-600 via-lime-700 to-lime-900">
  {/* 배경 패턴 */}
  <div className="absolute inset-0 opacity-10">
    <div className="rounded-full -translate-x-1/2 -translate-y-1/2" />
  </div>

  {/* 컨텐츠 */}
  <div className="relative">...</div>
</section>
```

**대표 프로필 섹션**:
```tsx
<div className="bg-gradient-to-br from-lime-50 to-white border border-lime-100">
  <div className="grid md:grid-cols-2">...</div>
</div>
```
