  - 포트폴리오 페이지 (주요 고객사 섹션)

### 필요한 로고 파일

현재 필요한 클라이언트 로고 (우선순위 순):

| 번호 | 회사명 | 파일명 | 상태 |
|-----|--------|--------|------|
| 1 | 삼성 (SAMSUNG) | `samsung.svg` 또는 `samsung.png` | ⏳ 대기 |
| 2 | CJ | `cj.svg` 또는 `cj.png` | ⏳ 대기 |
| 3 | 기아자동차 (KIA MOTORS) | `kia.svg` 또는 `kia.png` | ⏳ 대기 |
| 4 | 농심 | `nongshim.svg` 또는 `nongshim.png` | ⏳ 대기 |
| 5 | 미래에셋증권 | `miraeasset.svg` 또는 `miraeasset.png` | ⏳ 대기 |
| 6 | KB국민은행 | `kb.svg` 또는 `kb.png` | ⏳ 대기 |
| 7 | LG | `lg.svg` 또는 `lg.png` | ⏳ 대기 |
| 8 | 롯데 | `lotte.svg` 또는 `lotte.png` | ⏳ 대기 |

### 이미지 사양

**형식**:
- **권장**: SVG (벡터 이미지, 확대/축소 시 품질 유지)
- **대체**: PNG (투명 배경)

**크기**:
- **권장 크기**: 200px × 100px (가로 × 세로)
- **최소 크기**: 150px × 75px
- **최대 크기**: 400px × 200px
- **비율**: 2:1 (가로가 세로의 2배)

**배경**:
- 투명 배경 필수 (PNG 또는 SVG)
- 로고만 포함, 여백 최소화

**파일 크기**:
- SVG: 제한 없음 (일반적으로 10KB 이하)
- PNG: 50KB 이하 권장

**색상**:
- 원본 컬러 로고 (사이트에서 grayscale 필터 자동 적용)
- 또는 회색조 버전 제공 가능

### Next.js Image 최적화

현재 구현된 Next.js Image 컴포넌트는 다음 최적화를 자동 적용합니다:

- ✅ **자동 포맷 변환**: WebP 형식으로 자동 변환 (지원 브라우저)
- ✅ **반응형 이미지**: 화면 크기에 맞춰 적절한 크기 로드
- ✅ **지연 로딩**: 화면에 보일 때만 로드 (성능 향상)
- ✅ **Grayscale 효과**: hover 시 컬러로 전환

## 📝 이미지 제공 방법

### 1단계: 이미지 준비
- 위 사양에 맞춰 이미지 준비
- 파일명은 영문 소문자 + 숫자 (예: `samsung.svg`)

### 2단계: 파일 전달
- Slack, 이메일, 또는 클라우드 스토리지 링크로 전달
- 또는 직접 `/public/images/clients/` 폴더에 추가

### 3단계: 코드 업데이트
이미지 파일을 받으면 다음 파일만 수정하면 됩니다:

**포트폴리오 페이지** (`app/portfolio/page.tsx`):
```typescript
const keyClients = [
  { name: 'SAMSUNG', logo: '/images/clients/samsung.svg' }, // placeholder.svg → samsung.svg
  { name: 'CJ', logo: '/images/clients/cj.svg' },           // placeholder.svg → cj.svg
  // ...
];
```

**홈페이지 컴포넌트** (`app/_components/home/key-reference.tsx`):
```typescript
const clients = [
  { name: '삼성', logo: '/images/clients/samsung.svg', industry: '전자' },
  { name: 'CJ', logo: '/images/clients/cj.svg', industry: '엔터테인먼트' },
  // ...
];
```

## 🔍 참고사항

### 저작권 및 라이선스
- 모든 클라이언트 로고는 해당 회사의 공식 로고 사용 가이드라인 준수
- 웹사이트 게시 권한 확보 필수

### 성능 최적화
- Next.js Image 컴포넌트가 자동으로 최적화 수행
- 원본 이미지만 제공하면 자동으로 WebP 변환 및 크기 조정

### 업데이트
- 로고 변경 시 파일만 교체하면 즉시 반영
- 빌드 또는 배포 재시작 불필요 (정적 파일)
